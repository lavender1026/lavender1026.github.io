Linux学习

By DirkChen

發表於 2017-06-21

文章目錄 
因为这次使用Gbase的数据库，数据库的安装要放在Linux上，所以我这次算是第一次接触了Linux。第一次使用Linux，真的是好清新好清爽，没有了wps，淘宝广告，只有terminal和代码，爽的一批。然而，作为新人，指令也是需要牢牢记住很多的。这里简单的记录下一些简单的指令
cd ./当前目录 后面可以跟二级目录
ll 用于查看当前目录所含文件
ll -h 用于查看当前目录所含文件,并包含目录文件大小
df -h 查看Linux存储空间
netstat -ano|grep xxxx 查看xxxx端口是否被占用

vi 打开vim
i进入编辑模式
esc退出 ：wq保存并推出 q退出 q！强制退出
vi模式下/xxxx 快速检索xxxx字段
mv xx bb 重命名xx为bb
rm -rf /文件名
rm -f 直接删除
